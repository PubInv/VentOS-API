Please refer to the PIRCS standard for a detailed description of meaning of possible entries. The following values are a command (C) to set the Target (T) Pressure (P) to 25 cmH2O (we transfer integral number of cmH2O times 10, or mmH2O): 
{ "com": "C",
  "par" : "P",
  "int" : "T",
  "mod" : 0,
  "val" : 250
    }
To test this programmatically, use can use anything that makes a post such as curl: 
curl -d "com=C&par=P&int=42&mod=0&val=250" -X POST http://127.0.0.1:8000/control/
